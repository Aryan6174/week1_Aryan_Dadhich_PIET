# 🌍 Country Intelligence System

AI-Powered Country Segmentation Platform for HELP International

## 📋 Overview

This application helps HELP International identify countries in dire need of aid using machine learning algorithms including clustering and classification models.

## 🚀 Features

- **Data Analysis**: Comprehensive EDA with interactive visualizations
- **Clustering**: K-Means and DBSCAN algorithms
- **Classification**: 4 ML models (Logistic Regression, Decision Tree, Random Forest, XGBoost)
- **Recommendations**: Priority-based country ranking
- **Budget Allocation**: Strategic fund distribution tools
- **Export Results**: Download analysis in multiple formats
